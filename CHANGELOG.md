## [2.0.47] - 2026-04-24

### Added
- Initial open source release of Release Hub for Azure DevOps.
- Unified management of YAML Pipeline and Classic Release approvals.
- Advanced filtering by pipeline, repository, stage, and branch.
- Stage visualization with status indicators.
- Approval history tracking.
- Azure DevOps light/dark theme support.
- Release flow tracking with merge-base detection for hotfixes and cherry-picks.
- Optional tracking work item creation for releases.
- Application set grouping and DORA metrics views.
