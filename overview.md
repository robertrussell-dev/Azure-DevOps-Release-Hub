# Release Hub

Release Hub is an Azure DevOps extension for tracking and managing pending approvals across YAML Pipelines and Classic Releases in one place.

## Highlights

- Unified queue of pending approvals
- Advanced filtering by pipeline, repository, stage, and branch
- Stage flow visualization with status context
- Approval history and action tracking
- Azure DevOps theme-aware UI

## Getting Started

1. Install the extension from the Marketplace.
2. Open Azure DevOps and navigate to Pipelines > Release Hub.
3. Review pending approvals, apply filters, and take approve or reject actions.
